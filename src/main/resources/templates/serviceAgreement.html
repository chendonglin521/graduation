小米手机平板产品退换修政策

 

小米手机平板产品售后政策严格执行国家三包法，遵守行业准则，并充分尊重消费者权益，为消费者提供全方位服务保障和郑重承诺，对国家规定或者与消费者约定包修、包换、包退的商品，小米承诺负责修理、更换、或者退货，授权服务中心、寄修中心需参照本政策开展工作。

 

一、三包期说明

小米公司承诺由产品购买（签收）次日起质量问题7日内包退， 15日内包换，一年包修，网购商品享受7日内无理由退货

保修范围

保修内容

保修期限

整机

机器本身一体的零配件

1年

附件

充电器、数据线、电池等

备注：小米手机、平板产品以系统激活时间作为三包起始时间的判断依据（激活时间为用户第一次连接互联网使用小米手机的时间，包括但不仅限于WIFI、2/3/4G移动数据网络）

 

 

二、退货政策及流程

1.     自营线上渠道购买—小米商城、天猫旗舰店、淘宝自营店等

1）7日无理由退货

小米向您承诺：7日内，无人为损坏，包装配件齐全，可以无理由退货

2）7日故障退货

小米向您承诺：7日内，如出现功能性故障或商品质量问题，经小米公司授权服务网点检测，确认属于非人为商品质量问题，我们将免费会为您退货

3）申请售后方法

l  小米商城（网页、手机APP）等

小米手机如需申请退货，请在 ［我的订单－申请服务］ 或拨打400-100-5678申请售后服务，邮寄至小米公司办理退货服务

l  小米天猫旗舰店、淘宝自营店

小米手机如需申请退货，请在天猫、淘宝系统中申请售后服务，邮寄至小米公司办理退货服务

 

2.     自营线下渠道购买—小米之家、小米专卖店等

1）7日故障退货

小米向您承诺：7日内，如出现功能性故障或商品质量问题，经小米公司授权服务网点检测,确认属于非人为商品质量问题，请联系手机销售方办理退货

2）申请售后方法

方法一：手机办理故障退货，需要经小米公司授权服务网点检测符合退货标准后，联系手机销售方进行退货服务

方法二：小米手机如需申请退货，请在 ［我的订单－申请服务］ 或拨打400-100-5678申请售后服务，邮寄至小米公司办理退货服务（仅限非现金购买）

 

3.     非自营渠道

其他渠道购买手机办理退货，经小米公司授权服务网点检测符合退货标准后，按照销售方退货政策办理

 

4.     注意事项

1）小米之家和小米专卖店为线下销售，不支持7日无理由退货

2）以下情况将无法享受7日无理由退货

A. 进行未经授权的维修、改动，破坏、涂改强制性产品认证标志、指示标贴、机器序列号等

B. 有难以恢复原状的外观类使用痕迹，或者产生激活、授权信息、不合理的个人使用数据留存等数据类使用痕迹

3）7日无理由退货，请自付运费发快递至小米的寄修中心，不接受到付和平邮方式；若以到付和平邮方式返回，有可能会被拒收或认可从货款中扣除垫付的运费

4）7日内因质量问题，请自付运费发快递至小米的寄修中心，不接受到付和平邮方式；经检测后符合退货标准，运费将予以报销（最高每单15元），如不符合退货标准小米会将产品原地址寄回

5）消费者退货时应当将商品本身、配件（含包装、附件等）及赠品一并退回；赠品包括赠送的实物、积分、代金券、优惠券等形式。如果赠品不能一并退回，经营者可以要求消费者按照事先标明的赠品价格支付赠品价款。

6）以下情况将无法享受退货服务

l  产品已经超过退货期；

l  产品外观、功能（7日无理由退货可不参考）不符合退货标准

l  主机附带的机身铭牌如IMEI/IMEID标签、入网标签损毁或丢失，防拆标损毁或私自更换，防水标变色等

l  未经小米公司许可和授权，擅自拆卸、维修和改装（包括硬件或系统软件修改）

l  商品的外包装、附件、赠品、说明书不完整，发票缺失或涂改

l  因意外因素或人为原因（包括主观/非主观原因如浸水、震裂、摔坏、印刷线路板及功能器件烧坏等）导致移动电话故障或损坏

l  因不可抗力（如地震、火灾、雷击、电压不稳等）原因造成的移动电话故障或损坏

l  未经小米公司许可和授权，擅自拆卸、维修和改装（包括硬件或系统软件修改）

l  商品的外包装、附件、赠品、说明书不完整，发票缺失或涂改

l  因意外因素或人为原因（包括主观/非主观原因如浸水、震裂、摔坏、印刷线路板及功能器件烧坏等）导致移动电话故障或损坏

l  因不可抗力（如地震、火灾、雷击、电压不稳等）原因造成的移动电话故障或损坏

三、     维修政策及流程

（一）免费维修政策

小米向您承诺：1年内，如出现功能性故障或商品质量问题，经小米公司授权服务网点检测,确认属于非人为商品质量问题，我们将会为您免费维修

1.     申请方法

方法一：您也可以到访身边，具有维修功能的授权服务网点办理维修服务

方法二：小米手机如需申请维修，请在［我的订单－申请服务］或拨打400-100-5678申请售后服务，邮寄至小米公司办理维修服务

2.     注意事项

1）维修网点售后建议提前电话联系咨询需要携带的凭证、受理时间等

2）以下情况将无法享受免费维修政策

l  产品超出保修期

l  机身串码超出保修期，且无有效维修凭证或维修凭证异常的（包括无有效购机凭证、凭证上的内容与商品实物标识不符或有涂改的）

l  主机附带的机身铭牌如IMEI/IMEID标签、入网标签损毁或丢失，防拆标损毁或私自更换，防水标变色等

l  未经小米公司许可和授权，擅自拆卸、维修和改装（包括硬件或系统软件修改）

l  未按使用说明书操作或连接非小米公司配件造成手机损坏或功能异常的

l  因意外因素或人为原因（包括主观/非主观原因如浸水、震裂、摔坏、印刷线路板及功能器件烧坏等）导致移动电话故障或损坏的

l  使用中造成的除触摸屏外的外壳类元器件磨损、开裂、折断、破洞、烧焦、脱焊

l  因不可抗力（如地震、火灾、雷击、电压不稳等）原因造成的移动电话故障或损坏的

l  出现小米判定政策中《疑似非保问题判定标准》中描述的非保情况

1.     自费维修类型

小米向您承诺：您只需支付少许费用，手机可享受终身维修服务

2.     维修申请方法

方法一：您也可以到访身边，具有维修功能的授权服务网点办理维修服务

方法二：小米手机如需申请维修，请在［我的订单－申请服务］或拨打400-100-5678申请售后服务，邮寄至小米公司办理维修服务。
